Pastelería Web React

Tecnologías utilizadas

  • React 18
  • React Router DOM
  • Vite
  • ESLint

Descripción

Aplicación web desarrollada en React para una pastelería. Incluye catálogo de productos, carrito de compras, resumen de compra, contacto y secciones informativas.

El proyecto usa rutas con React Router y una estructura modular basada en componentes reutilizables.

Sistema de Gestión de Biblioteca

Tecnologías utilizadas

  • Java (Spring Boot)
  • Spring MVC / REST Controllers
  • Thymeleaf o JSP
  • Base de datos relacional
  • Arquitectura MVC

Descripción

Aplicación web desarrollada con Spring Boot para gestionar bibliotecas. Permite el registro y control de libros, autores, editoriales, stock y préstamos.

Incluye módulos de reportes, controladores REST, entidades JPA y una estructura organizada para su mantenimiento y escalabilidad.

Restaurante Acurio’s – Carta Virtual

Tecnologías utilizadas

  • Java (JSP + Servlets)
  • HTML, CSS, Bootstrap
  • MySQL
  • JDBC (Conexión a BD)

Descripción

Sitio web dinámico para un restaurante peruano. Permite gestionar platos, ver el catálogo gastronómico, realizar reservas y acceder según tipo de usuario (cliente, administrador, empleado).

Usa sesiones, Bootstrap y una base de datos con tablas para usuarios, platos, pedidos, reservas y más. Ideal como prototipo funcional para un restaurante real.
